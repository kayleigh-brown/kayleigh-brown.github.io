Projects

$aveur

Backend Developer

A website designed to help users learn to budget and save money. A user can implement limits to certain categories of spending and $aveur will keep track of how much budget they have remaining within a certain category. If by the end of the month they have a surplus within a category, that surplus is automatically deposited into a savings account of the user's choosing. I was responsible for creating a rudimentary back-end to the website to keep track of bank accounts, spending history, and budget categories for multiple users. I worked with Nessie, Capital One's Hackathon API in order to connect with the user's bank account to track categorical spending.

Won Best Financial Hack at the WiCS Spring 2019 Hackathon
